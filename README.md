## 🎯 The Mission
Join our community to master C/C++, Logic Building, and Professional Git Workflow.

---

### 📌 How to Participate (Rules):
1. **Fork** this repository to your account.
2. **Clone** the forked repo to your local machine.
3. Create a folder with your name: `Name_Solutions` (e.g., `Dev_Solutions`).
4. **Daily Task:** Create a sub-folder for each day (e.g., `Day01_Calculator`).
5. **Commit & Push** your code to your fork.
6. Send a **Pull Request (PR)** daily for review.
